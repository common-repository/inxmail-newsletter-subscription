The plugin enables you to create a subscription-form for your newsletters, if you use inxmail as your email-marketing-tool.

== Description ==

The Plugin gives you options to create a subscription-form for your newsletters based on Inxmail.
You can:

* create you own individual form
* integrate the form in any page or article via shortcode
* use a quick-subscription as a widget (only email-address)
* configure your inxmail-list, where to subscribe
* select list from available set of lists, if you connect with inxmail via api


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/inxmail-newsletter-anmeldung` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Inxmail -> Settings screen to configure the plugin
4. Create your own subscription form through Inxmail -> Formular bearbeiten

== Frequently Asked Questions ==

= Can I create a form with this plugin? =

Yes, you can!
It can have individual fields.
You can also use a quick-subscription as a widget with only the email-address.

= How can I use the login area under Settings? =

You must request the API from your Inxmail partner and paste it into the following plugin folder:
"inxmail-anmeldung\inxmail\inxmail_api\"
